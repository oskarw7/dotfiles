# Notes

## 1. TPM:
    - TPM needs to be installed manually with `git clone https://github.com/tmux-plugins/tpm ~/.config/tmux/plugins/tpm`, then press `<C-space>I`

## 2. Rose-pine:
    - during installation of this plugin TPM overrites config
    - it needs to be replaced with custom config by `cp ./themes/rose-pine.tmux ./plugins/tmux/rose-pine.tmux`
